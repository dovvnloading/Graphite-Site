All Features

Canvas & Navigation

  • Infinite Node-Graph Canvas
  • Pan (Middle Mouse, WASD) & Zoom (Ctrl+Wheel, Q/E)
  • Area Zoom (Shift+Drag)
  • Conversation Timeline Minimap
  • Navigation Pins & Bookmarks
  • Sticky Notes for Annotations (Ctrl+N)
  • Dynamic Grid with Snap-to-Grid
  • Smart Alignment Guides
  • Orthogonal Connection Routing

Content & Nodes

  • Multi-modal Chat Nodes (Text, Images, Docs)
  • Code Nodes with Syntax Highlighting
  • Image & Document Attachment Nodes

  • Scrollable & Collapsible Node Content

  • Document Viewer Panel for Focused Reading
Create diverse content types, from markdown-rich text to code blocks and file attachments, all visually organized on the canvas.

Knowledge Synthesis

  • Image Generation from Text Prompts
  • Custom System Prompts per Branch
Turn raw information into insight. Summarize single nodes, synthesize multiple nodes into a cohesive overview, and visualize data on the fly.

Plugins & Extensibility

  • **Py-Coder:** AI-driven & Manual Code Execution
  • **graphite-Web:** Context-Aware Web Research
  • **Reasoning Engine:** Multi-step Problem Solving
  • **Conversation Node:** Encapsulated Sub-chats
  • Plugin Architecture for Custom Tools
  • Command Palette for Full Access (Ctrl+K)
Extend Graphite with powerful plugins like a code interpreter (Py-Coder), a web researcher, a reasoning engine, and more.

Connections & Relationships

Session & Knowledge Base
