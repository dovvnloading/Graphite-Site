Overview

Graphite Pro is a secure, local-first AI workspace that merges human intuition with model intelligence. It replaces linear chat with a node-graph canvas, so you can branch ideas, compare approaches, and keep an auditable trail of how you think. The result is a workflow that’s faster to explore, easier to explain, and fundamentally more creative — all while keeping your data on your machine.

FAQ

Is Graphite open-source?

The code is public and community-driven. A lightweight license + trial keeps development sustainable while preserving the open ethos.

Do you collect data?

No telemetry. Local models run on your machine. API usage remains between you and your provider.

How does licensing work?

Install → 5-day full-feature trial → activate with a key in-app. Keys are validated locally; no always-online checks.

Which models are supported?

Ollama (local), plus APIs for OpenAI, Anthropic, Gemini, Groq, and other OpenAI-compatible endpoints.

Can I contribute?

Yes. PRs are welcome. Early adopters help shape plugins, UX, and roadmap priorities.
