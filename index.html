The Integrated AI Workspace

Go beyond linear chat. Graphite is a visual canvas for complex thinking, turning your conversations into structured knowledge graphs. Run code, search the web, and synthesize ideas—all in one private, local-first environment.

Why Graphite?

Think Visually

Branch conversations, connect disparate ideas, and build non-linear thought processes. The node-graph UI lets you see the full context of your work at a glance.

Absolute Privacy

Your data is yours. Run powerful models completely offline with Ollama. No tracking, no telemetry, and zero data leaves your machine unless you connect to an external API.

Connect Any Model

Switch seamlessly between local models (Ollama) and best-in-class APIs from OpenAI, Anthropic, Google Gemini, Groq, and any OpenAI-compatible endpoint.

A True Workbench

Execute code with the Py-Coder, perform live web research with the Web Agent, and solve complex problems with the Reasoning Engine. Graphite is a workspace, not just a chat app.

Build Your Knowledge

Save entire reasoning sessions as persistent, searchable knowledge graphs. Your work compounds over time, creating a second brain that grows with every project.

Synthesize & Understand

Use built-in AI tools to generate takeaways, explain complex topics, or create synthesized summaries from multiple nodes. Turn raw information into structured insight.

An Integrated AI Toolkit

Code Interpreter

Generate, execute, and debug Python code with an AI-driven agent. Switch to manual mode to run your own scripts in a persistent environment.

Web Researcher

Perform context-aware web searches. The AI refines your query, validates sources for safety and relevance, and synthesizes the findings into a clean summary.

Reasoning Engine

Tackle complex problems with a multi-step "Chain of Thought" agent that plans, reasons, critiques its own work, and synthesizes a final, robust answer.

Knowledge Synthesizer

Distill insights from your work. Generate one-click explainers, takeaways for a single node, or create a synthesized summary that finds the connection between multiple nodes.

Conversation Manager

Encapsulate long, linear discussions into a single, self-contained Conversation Node. Keeps your main canvas clean while preserving deep contextual history for sub-tasks.

Data Visualizer

Generate charts and graphs directly from text. Right-click any node with data to instantly create bar charts, line graphs, pie charts, and more.

Supported Models & Providers

Run locally with Ollama or connect to frontier models via API. Everything routes through Graphite's unified node-graph interface.

Ollama

Run models locally. Fully offline, privacy-first. No API keys. Complete control and zero data transit.

OpenAI

GPT-class models via API. Advanced reasoning and multi-modal capabilities for frontier inference.

Anthropic

Claude models optimized for extended thinking, analysis, and nuanced reasoning tasks.

Google Gemini

Multimodal capabilities including vision, text, and advanced understanding. Direct API integration.

Groq

High-speed inference for real-time, interactive workflows. Low-latency reasoning and response generation.

OpenAI-Compatible

OpenRouter, Together, self-hosted models, and any endpoint that supports the OpenAI protocol.
