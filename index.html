Complete Feature List

📋 Canvas & Navigation

🎨 Node-Graph Canvas
Organize reasoning spatially. Branch paths, fork decisions, and maintain clarity about how you arrived at conclusions.
🖱️ Intuitive Navigation
Pan with middle-mouse, zoom with Ctrl+scroll, area zoom with Shift+drag. Rubber-band selection and multi-item manipulation.
📌 Frames & Grouping
Group multiple nodes into movable containers. Resize, lock, customize with colors and notes for organization.
📍 Navigation Pins
Bookmark specific canvas locations. Name, annotate, and jump to saved positions across your workspace.
✏️ Sticky Notes
Floating annotations with rich-text editing. Resizable and color-customizable for visual organization.
🔲 Smart Grid System
Dynamic, zoom-responsive grid with customizable opacity and size. Optional snap-to-grid and smart alignment guides.

💬 Content & Nodes

💬 Chat Nodes
Multi-modal support with formatted text (Markdown) and images. Scrollable content, collapsible headers, and contextual parent relationships.
{"} Code Nodes
Syntax highlighting with automatic language detection. Copy-code button, parent linking, and full code exploration.
📄 Document Nodes
Display attached file content with scrollable viewing. Automatic text extraction from supported formats.
🎯 Contextual Branching
Generate multiple responses from any point in the conversation. Create exploratory branches and compare approaches effortlessly.
🔄 Response Regeneration
Request new AI responses for any prompt. Replace existing nodes and explore alternative reasoning paths.
📎 Multi-Modal Input
Include images in prompts for models that support vision. Seamless image attachment via drag-and-drop or file picker.

🔗 Connections & Relationships

➡️ Conversation Flow
Visualize chronological and logical conversation branches. Color-graded lines with pulsing arrow animations on hover.
🎚️ Custom Path Routing
Add pins to connections with Ctrl+Left-Click to create bezier curve control points. Move or remove pins for precise path customization.
🔗 Content Links
Visualize relationships between chat nodes and generated code or document nodes with distinctive dashed styling.
⭐ System Prompt Links
Special pulsing connections linking system prompt notes to conversation branches for custom AI behavior.
📐 Orthogonal Routing
Automatic right-angle path routing for connections. Keeps your canvas organized and easier to follow.

🤖 AI & Content Analysis

💡 Multi-Provider Support
Run the same workflow across OpenAI, Anthropic, Gemini, Groq, Ollama, or any OpenAI-compatible endpoint.
📊 AI Chart Generation
Analyze text content and auto-generate charts. Supports bar, line, pie, histogram, and Sankey diagrams.
🎨 Image Generation
Generate images from text prompts using Gemini's imagen model. Results appear as new chat nodes.
📌 Generate Takeaway
Create concise summaries of any node's content as searchable sticky notes.
📚 Generate Explainer
Generate simplified, high-level explanations of complex content as standalone notes.
🎭 System Prompt Customization
Override default system prompts for entire branches. Enable role-based AI behavior like code review or writing coaching.

📁 File & Content Handling

📤 File Attachment
Attach files via dedicated button or drag-and-drop. Automatic parsing and visualization in document nodes.
🖼️ Image Support
PNG, JPG, JPEG, WebP formats for multi-modal AI prompting and inline visualization.
📋 Document Formats
TXT, MD, PY, JSON, HTML, CSS, JS, CSV, XML, PDF, DOCX. Automatic text extraction and proper formatting.
🔍 Full-Text Search
Find text across any node on the canvas. Navigate with next/previous buttons for comprehensive content discovery.
✅ Spell Checking
Built-in spell check in the main text input field with context menu suggestions.

💾 Session & Knowledge Management

📚 Chat Library
Central dialog for managing all saved sessions. Load, delete, rename, and search by title with Ctrl+L.
💾 Manual Save
Save entire canvas state to local SQLite database with Ctrl+S. Complete state preservation including zoom and scroll position.
🗂️ Persistent Projects
All nodes, connections, frames, notes, pins, and charts are saved. Revisit and build on past work indefinitely.
🏷️ Automatic Titling
AI-generated concise titles based on initial conversation content when saving new chats.
🔐 Local Storage
All data stays on your device. No cloud sync, no external dependencies, complete data ownership.

⚙️ Configuration & Models

🌐 Ollama Integration
Run models locally with full offline capability. Model selection dialog and automatic validation.

Configure OpenAI, Anthropic, Gemini, Groq, OpenRouter, and self-hosted endpoints with flexible model assignment.

🔌 OpenAI-Compatible Endpoints
